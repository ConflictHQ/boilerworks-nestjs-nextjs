# Security Policy

## Reporting a Vulnerability

Please do **not** open a public issue for security vulnerabilities.

Instead, report them privately via GitHub Security Advisories:

https://github.com/ConflictHQ/boilerworks-nestjs-nextjs/security/advisories/new

We will acknowledge your report as soon as possible and keep you informed of
progress toward a fix.

## Supported Versions

This is a template repository. Only the latest state of `main` is supported —
projects generated from this template are responsible for their own dependency
updates after cutting.
